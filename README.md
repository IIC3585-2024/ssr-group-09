The application allows users to see a list of series and movies.

## Features

- *See Series*: Users can see events series and events
- *Create serie o movie*: Users can create a new serie o movie
- *Add comment and ratings*: The application is designed to add comments and ratings for the actual series and movies


## Technologies

This project is built using the following technologies:

- *Next*: Used for the application logic and user interaction.

## Installation

To install and run this project, follow these steps:

1. Clone the repository: git clone https://github.com/IIC3585-2024/ssr-group-09
2. Navigate to the project directory: cd ssr-group-09
3. Install the dependencies: npm install
4. Star fake api: npx json-server db.json --port=3001
5. Start the development server: npm run dev

Open [http://localhost:3000/](http://localhost:3000/) to view the application in your browser.
